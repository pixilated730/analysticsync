## AnilistSync
Sync anilist with anidb & mal, mapping gogoanime sub & dub identifiers

Example:

```
https://raw.githubusercontent.com/kruncx/anilistsync/main/data/1.json
```

Response:
```
{
    "id": 1,
    "anidb": 23,
    "myanimelist": 1,
    "gogoanime": [
        "cowboy-bebop",
        "cowboy-bebop-dub"
    ]
}
```
